<img src="icon/app-logo_512.png" alt="Extension Icon" height="32"/> Movie Button for IMDb
========================

A Chrome extension that integrates directly with IMDb to help you find movies and TV shows on alternative streaming services, databases, and torrent trackers.

## Features

-   **Seamless IMDb Integration**: Adds customizable "Find on Rezka", "Find on [Platform]", and "Find on [Tracker]" buttons directly to the IMDb movie page (below the "Add to Watchlist" button) to open a new tab with search results.
-   **Full Control on Popup**:
    -   **Toggle Functionality**: Independently enable or disable the Rezka, Streaming/Database, or Torrent buttons via the popup settings.
    -   **Rezka Mirror**: Select from a list of Rezka mirrors (e.g., `rezka.ag`, `hdrezka.tv`).
    -   **Streaming & Database**: Select from a list of streaming services and databases (e.g., JustWatch, Megogo, Rotten Tomatoes, Kinorium).
    -   **Torrent Tracker**: Select your preferred torrent tracker (e.g., `toloka.to`, `rutracker.org`).
    -   **Edit Lists**: Easily add or remove search links and domains of all three buttons to keep up with active resources.

## Installation

1.  Clone or download this repository.
2.  Open Chrome and navigate to `chrome://extensions`.
3.  Enable **Developer mode** in the top right corner.
4.  Click **Load unpacked**.
5.  Select the `Movie-Button` folder.
